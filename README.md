# Ayano (WIP)
Ayano is Open Source Blog Engine.

- HTML5 support
- use [bootstrap](https://github.com/twbs/bootstrap)

- WordPress Blog import (WIP)

## requirements

- Node.js (>=7.0.0 ?)
- npm
- MongoDB Server

## How to run

```
npm install
gulp build
node dest/main.js
```
